# BigDataSnowflake

Анализ больших данных - лабораторная работа №1 - нормализация данных в схему Snowflake

## Запуск

```bash
git clone https://github.com/mxdesta/BDSnowflakelab.git
cd BDSnowflakelab
docker compose up --build -d
```

## Подключение к БД

- Host: `localhost:5432`
- Database: `bigdata_snowflake`
- User: `bigdata_user`
- Password: `bigdata_password`

БД автоматически заполняется при запуске (около 30 секунд).
